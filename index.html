Core Concepts

核心概念与定义

非量测摄影机

光学畸变差较大且无框标装置，但价格低、灵活性强，适合遥控、水下、高速等特殊场景。

摄影机主距 / 焦距 f

摄影机物镜中心到像片面的固定距离，是决定比例尺、视场角和航高设计的核心参数。

像片倾角 α

摄影瞬间主光轴与铅垂线的夹角。竖直摄影通常要求倾角小于 2°~3°，倾斜摄影则常用于城市三维重建。

地面分辨率 GSD

影像上单个像素在地面代表的尺寸，单位一般为米/像素，是成图质量与航摄设计之间的关键桥梁。

相对航高与绝对航高

相对航高以地面平均高程基准面为参考，绝对航高则相对于平均海平面，两者满足 H绝 = H + H地。

航摄高程要求

实际飞行时相对航高与设计值相差不得大于 5%，同一航线内相邻摄站高差一般不超过 50 m。

航向重叠度 px

同一航线相邻像片的重叠，通常要求 60%~65% 以上，以保证复杂起伏地区仍能满足立体测图需要。

旁向重叠度 py

相邻航带像片之间的重叠，一般控制在 30%~35%，地形起伏越大越需要适当增加。

三度重叠

航向上三张相邻像片拥有公共重叠区，是连续立体观察和像片连接的必要条件。

立体像对与摄影基线 B

两张航向相邻像片构成立体像对，相邻摄影站之间的空间距离称为摄影基线。

航带弯曲度

衡量航带偏离设计直线程度的重要指标，通常要求最大偏距比不大于 3%。

像片旋偏角 κ

描述像片相对航带方向的旋转，一般规定不超过 6°，过大时会显著削弱有效重叠区域。

Key Formulae

重要公式

相对航高公式

H = (f × GSD) / a

其中 H 为相对航高，f 为焦距，GSD 为地面分辨率，a 为像元大小。

摄影比例尺公式

1 / m = l / L = f / H

影像距离与地面距离的比值等于焦距与航高之比，因此也可写成 H = m f。

绝对航高公式

H绝 = H + H地

绝对航高由相对航高与地面平均高程相加得到。

航向重叠度

Px = (px / lx) × 100%

用重叠部分边长与像幅边长之比表达航向重叠程度。

旁向重叠度

Py = (py / ly) × 100%

用相邻航带公共覆盖宽度与像幅宽度之比表达旁向重叠程度。

Projection Basics

中心投影与正射投影

投影的基本定义

投影是使用一组假想的直线，也就是投影射线，将物体投射到投影平面上，并在该平面形成构像的过程。

投影的三大分类

中心投影：所有投影射线汇聚于同一个投影中心，航摄像片属于这种形式，存在变形且比例尺处处不等。

平行投影：所有投影射线平行于某一固定方向。

斜投影 / 正射投影：斜投影是射线与投影面斜交的平行投影；正射投影则要求射线平行且垂直于投影面，地图属于正射投影。

为什么地形图比例尺一致

正射投影中投影射线互相平行且与投影平面正交，因此图上各处比例尺一致、几何关系稳定，不会像中心投影那样随着位置和高差变化而改变。

Key Differences

航摄像片与地形图的六大差异

投影方法

航摄像片属于中心投影，地形图属于正射投影。

误差来源

像片误差主要来自像片倾斜和地形起伏；地形图误差主要来自测量精度和绘图精度。

比例尺特性

航摄像片比例尺处处不一致；地形图比例尺处处一致。

表示形式

像片呈现真实影像；地形图属于线化图。

表示内容

像片是客观成像，保留大量细节；地形图经过人为取舍，使用专门符号和注记表达内容。

几何特性

两幅相邻航摄像片可构成立体像对，而单张地形图不具备此特性。

Chapter 3-4

共线方程

Part 1

摄影测量常用的坐标系

物方与像方解析关系基础

核心任务是建立地面点与像点之间的解析联系。常用的五种坐标系如下：

像平面坐标系 (o-xy)

表示像点在像平面上的位置。可由物理框标建立，也可取像主点为原点。

像空间坐标系 (S-xyz)

右手系，原点在投影中心 S。像点坐标通常记为 (x, y, -f)。

像空间辅助坐标系 (S-XYZ)

右手系，原点仍在投影中心，用于计算中的过渡与统一表达。

摄影测量坐标系 (A-XpYpZp)

右手系，原点取地面某点 A，是像空间辅助坐标系与物空间坐标系之间的过渡坐标系。

物空间坐标系 (O-XtYtZt)

左手系，用于描述地面物体所在空间，通常与大地坐标系对应。

Part 2

影像的内外方位元素

3 个内方位 + 6 个外方位

内方位元素

用于确定镜头中心与影像平面的相对位置，包含 3 个参数：

  • 像主点相对影像中心的位置 (x0, y0)
  • 主距或焦距 f

外方位元素

用于确定摄影瞬间摄影光束的空间位置和姿态，共 6 个参数：

  • 线元素：(Xs, Ys, Zs)
  • 角元素：(φ, ω, κ)

线元素含义

描述摄影中心 S 在物方空间坐标系中的位置，是空间平移参数。

角元素含义

φ-ω-κ 系统常用于表示姿态：φ 为航向倾角，ω 为旁向倾角，κ 为像片旋角。

Part 3

空间直角坐标系的旋转变换

正交变换

同原点坐标系之间可通过依次绕三个坐标轴旋转实现转换，旋转矩阵 R 具有正交矩阵性质。

重要特性

R 的行列式为 1，同行或同列元素平方和为 1，不同行或不同列对应元素乘积之和为 0。

正变换公式

[X Y Z]^T = R [x y -f]^T

逆变换公式

[x y -f]^T = R^-1 [X Y Z]^T = R^T [X Y Z]^T

旋转矩阵形式

R = [a1 a2 a3; b1 b2 b3; c1 c2 c3]

Part 4

中心投影的构像方程（共线条件方程）

摄影测量核心公式

摄影瞬间，像点、对应地面点与投影中心始终三点共线。这一关系是单像空间后方交会、光束法平差等算法的基础。

共线条件方程

{
x - x0 = -f a1(XA - XS) + b1(YA - YS) + c1(ZA - ZS) a3(XA - XS) + b3(YA - YS) + c3(ZA - ZS)
y - y0 = -f a2(XA - XS) + b2(YA - YS) + c2(ZA - ZS) a3(XA - XS) + b3(YA - YS) + c3(ZA - ZS)

参数说明

(x, y)：像点 a 在像平面直角坐标系中的坐标

(XA, YA, ZA)：像点对应物点在地面坐标系中的坐标

(XS, YS, ZS)：投影中心 S 在地面坐标系中的坐标，即外方位线元素

(a_i, b_i, c_i)：9 个方向余弦，包含 3 个外方位角元素

(f, x0, y0)：像片的内方位元素

Chapter 5

像点位移

围绕航摄像片中的两类主要位移展开：像片倾斜引起的像点位移 δα 与地形起伏引起的像点位移 δh。拖动参数可观察位移大小、方向及其几何中心的变化。

计算公式

δα = - rc2 f sin φ sin α
δh = na - na0 = rh H
Δh = Rh H - h

Part 1

像片倾斜引起的像点位移 (δα)

基本定义

在地面平坦条件下，由于像片倾斜，实际成像点相对于理想水平像片上的成像点发生偏移。

几何中心

倾斜位移发生在以等角点 c 为中心的辐射线上，主纵线方向位移最大。

位移规律

当 φ = 0° 或 180° 时，δα = 0；当 φ ∈ (0°, 180°) 时，像点向等角点方向位移；当 φ ∈ (180°, 360°) 时，像点背离等角点位移。

几何表现

地面上的正方形在倾斜像片上会变成不规则四边形，说明局部形状和比例关系发生变化。

Part 2

地形起伏引起的像点位移 (δh)

基本定义

即使像片完全水平，只要地面点相对基准面存在高差 h，其构像也会偏离理想位置，这种位移通常称为投影差。

关键参数

h 为地面点到基准面的高差，H 为摄影航高，r 为像点到像底点 n 的距离。

位移特性

投影差发生在以像底点 n 为中心的辐射线上，高差越大、离像底点越远，位移越显著。

地面投影差

除像片上的 δh = rh/H 外，地面上的投影差可写为 Δh = Rh/(H-h)，用于联系像片量测与地面影响。

Combined Effects

综合影响与其它因素

比例尺不一致

倾斜和地形起伏会使像片上各点位移不同，因此像片比例尺处处不等。

系统误差源

除两类主要位移外，物镜畸变、大气折光、地球曲率及 CCD 变形等也会引起规律性像点位移，通常通过数学模型改正。

Chapter 6

单像空间后方交会

利用单张影像范围内控制点的地面坐标与像点坐标，根据共线条件方程反求摄影瞬间的 6 个外方位元素，即 Xs, Ys, Zs, φ, ω, κ。

核心定义

已知：内方位元素 (x0, y0, f)、控制点物方坐标 (X, Y, Z)、像点观测值 (x, y)。

待求：6 个外方位元素 Xs, Ys, Zs, φ, ω, κ。

理论基础：共线条件方程。每个控制点提供 2 个方程，因此至少需要 3 个控制点。

计算流程

  1. 输入原始数据并归算像点坐标。
  2. 给外方位元素设置初值，常见做法是将旋转角初值取 0、坐标初值取控制点均值。
  3. 组成旋转矩阵 R，计算像点近似值及观测残差。
  4. 对共线方程进行泰勒展开并线性化，逐点建立误差方程。
  5. 按最小二乘原理解法方程，求未知数改正数。
  6. 判断是否收敛，未收敛则更新参数继续迭代。

后方交会示意图

Linearization

数学模型与线性化处理

非线性本质

共线方程本身是非线性的，因此需要通过泰勒级数展开并保留一次项，将其转化为可求解的线性误差方程。

误差方程

线性化后可写成类似 vx = a11 dXs + ... - lx 的形式，再统一组成法方程。

迭代求解

由于线性化依赖初值，求解需要逐步趋近，直到改正数小于限差为止。

最小二乘

通过最小二乘原理解法方程，可以同时利用多余观测，提高解算稳定性与可靠性。

Accuracy

精度评估与注意事项

单位权中误差

m0 = ±sqrt([vv] / (2n - 6))，其中 n 为控制点个数，可用于评价整体平差质量。

外方位元素精度

可由法方程系数矩阵逆阵 (A^T A)^-1 的主对角线元素计算外方位元素中误差。

控制点配置要求

控制点应避免落在同一圆柱面上，否则解可能不唯一；工程上常在像片四角布设 4 个控制点以提高可靠性。
